# Website tổng hợp CLB THPT Hòn Gai

## Mô tả
- Trang web có 8 CLB mô phỏng.
- Sử dụng thư mục `pic/` để chứa ảnh (8 ảnh đề xuất: `clb-tech.jpg`, `clb-photo.jpg`, ..., `clb-social.jpg`).
- Có animation, thiết kế hiện đại, responsive.

## Cách dùng
1. Copy ảnh của mỗi CLB vào thư mục `pic/`.
2. Đổi tên ảnh tương ứng với trường `img` trong `script.js`.
3. Mở `index.html` trên trình duyệt.

## Lưu ý
- Nếu ảnh không tồn tại, sẽ hiện ảnh placeholder tự động.
- Thay đổi màu sắc, kiểu chữ trong `styles.css` nếu cần.
